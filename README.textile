h1. App trivia-mobile http://trivia-mobile.herokuapp.com/
Descripción: Esta aplicación muestra una serie de preguntas y nos permite selccionar una respuesta. Si en el primer intento la respuesta ingresada es correcta tendrá un puntaje de 10 pts., si en el segundo intento la respuesta ingresada es correcta tendrá un puntaje de 5ts. y luego del tercer intento la respuesta tendrá un puntaje de 1 pt.

Para la construcción de esta aplicación se usaron jquery-mobile, jquery, middleman, parse, backbone.js y coffeescript.

h3. Los requisitos previos: 
* *Debe tener instalado ruby 1.87 -p 352*

h3. Pasos para poner en funcionamiento la app trivia-mobile:

1) Instalar Middleman (Middleman es una herramienta de línea de comandos para la creación de páginas web estáticas para más información visitar http://middlemanapp.com/)
Ir a la consola de comandos y escribir: 
<pre><code>
gem install middleman
</code></pre>
Esto instalará Middleman, sus dependencias y las herramientas de línea de comandos para su uso.
Este proceso agregará un nuevo comando, con 3 funciones útiles:
* *middleman init*
* *middleman server*
* *middleman build*

2) Iniciar el servidor de middleman:
En la consola escribir:
<pre><code>
cd trivia-mobile (Ir al directorio trivia-mobile)
trivia-mobile > middleman server –p 4567 
</code></pre>

3) Abrir el navegador y escribir la dirección localhost:4567 para ver si se vizualiza la app correctamente.

4) Crear cuenta en Parse https://parse.com/ 

5) Crear una app en Parse, luego ir a Dashboard y seleccionar la app creada anteriormente.

6) Copiar Application ID y REST API Key y pegarlos en los archivos formulario.js, vista_puntaje.js y trivia.js 
<pre><code>
	$.parse.init({
            		app_id : 'Application ID', 
            		rest_key : 'REST API Key'  
        		});
</code></pre>
7) Ir a Data Browser y elegir la opción Import existing data from a CSV file. En Name of new class poner preguntas_oficiales y seleccionar el archivo preguntas_oficiales.txt y luego Import. Al hacer esto se creará una tabla con los datos de las preguntas de la app.

8) Finalemente ir al navegador nuevamente y escribir localhost:4567 para probar la aplicación.
